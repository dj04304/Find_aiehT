## 📜 프로젝트 개요 및 목표 📜​
> - **프로젝트 명 : Find Aieht**  
> - **장르 : 액션 RPG, 경영 시뮬레이션 타이쿤**  
> - **2D/3D : 3D**  
> - **스토리 라인**  
마을 식당을 운영하던 식당 주인이 키우던 동물   
Aieht이 어느날 알 수 없는 이유로 인해 사라져 버렸고,   
동시에 다른 가축들도 포악하게 변해버렸다.   
이로인해 가게 운영이 힘들어졌고,   
소중한 친구 Aieht을 찾기위해 마을 밖으로 나가게 되는데…
